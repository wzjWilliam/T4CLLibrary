# T4CLLibrary
一个用于脱离电子教室控制的库，可以方便的做出用来在信息课上脱离控制的的程序，支持极域电子教室、机房管理助手和红蜘蛛电子教室。

## 注意：
- 本程序为**库**，而不是可执行文件（DemoGUI仅是演示，在信息课环境下使用并不方便）。
- 本人是C#新手，所以有些代码会十分奇妙，遇到问题可以反馈。

## 目的
方便想做这类程序的人不用重复造轮子，可以直接利用/参考这个库实现一些基础的脱离电子教室的功能

## 支持:

### 极域：
- 结束、挂起、恢复主进程
- 获取与设置密码
- UDP重放攻击
    - 发送消息
    - 发送命令（支持2016&2021版）
    - 远程关机和重启
- 开网
- 窗口化极域电子教室广播
### 红蜘蛛：
- 结束、挂起、恢复主进程
- 自动计算机房管理助手随机进程名
- 红蜘蛛UDP重放攻击
    - 发送命令
    - 发送黑屏与解除黑屏数据
    - 发送关机与重启命令
    - 发送允许和禁止聊天的命令
    - 发送要求签到与响应签到数据
    - 发送心跳数据
### 机房管理助手：
- 结束进程
- 设置机房管理助手密码和获取机房管理助手9\.99版及以上的的密码(获取为暴力破解)；
- 获取机房管理助手临时密码（仅支持部分版本）

### 其他：
- 更方便的结束、挂起或恢复进程的方法；
